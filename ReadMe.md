Selection Picker
Provide a list of item and it will select based on them.
It receives following props.

backgroundColor : string,
iconBackgroundColor : string,

format of searchItem, imageIcon is optional

#searchItem = [],

borderColor : string,

iconColor : string,

containerStyle : Stylesheet styles,

textStyle : Stylesheet styles
